# 함께사는양평 개발 기록

## 1. 위치기반안내 (/construction)
- 탭: 국가유산 → 풍경 → 동네가게 → 집으로 → 교통 → 알림
- **국가유산**: 문화재청 API, GPS 5km 반경, AI추천링크(집근처=내용, 관광=길찾기), 스탬프(방문인증), 대중교통/자전거/도보 링크, iframe 외부콘텐츠
- **풍경**: 같은 리+같은 계절 공유마당 사진
- **동네가게**: 같은 리 공유마당 승인 가게, 클릭→상세페이지
- **집으로**: 현재 GPS → 기본주소, 막차환승역 제안, 최적출발시간 계산, 알람
- **교통**: 교통돌발+지정체+도로공사 통합 (API 대기 중)
- **알림**: VillageAlert 모델, 마을 긴급정보, 관리자 CRUD

## 2. 회원정보 (/user/<id>)
- 3그룹 레이아웃: 프로필헤더 + 쪽지/벗 + 게시글목록
- 통벗 버튼: 🤖 통벗-이름, 성장·감정 표시
- 💬대화·✍️글쓰기·👥채팅·📅일정 팝업 연결
- 닢(포인트) 표시, 클릭→내역, 충전(/mypage/points/charge)
- 접속방법 표시 (구글/네이버/카카오/함사양)
- 현재위치: 로그인시 1회 GPS 수집, 수동갱신 버튼

## 3. 통벗(TongBot) AI 비서 (/user/my)
- **AI 채팅**: Groq API, 함께사는양평 플랫폼 지식 주입
- **성장 시스템**: 7단계(알→수호자), 경험치, 친밀도
- **감정 표현**: 💕따스한 🥲대견한 💪응원 😌걱정 😊기쁜 🙏축복
- **말투 선택**: 친근/존중/엄격
- **글쓰기**: 편집기(B/I/U·목록)·사진·파일·그리기, 임시저장→AI교정→게시
- **재능발견**: 대화 분석→숨은 재능 발견
- **상담연계**: 법률/심리 고민 감지→전문가 연결 제안 (닢 사용)

## 4. 벗 채팅 (/chat 팝업)
- 벗목록→선택→⚡바로채팅/📅예약채팅
- FriendCache DB 테이블로 빠른 로딩
- 통벗 중재: 비방감지·분위기조율
- 2시간 제한: 10분전 경고, 30초 카운트다운, 자동종료
- 초대 쪽지: HTML 버튼(입장/거절/모니터링)

## 5. 일정 관리 (/schedule 팝업)
- 달력 뷰, 추가/삭제
- 🔍 빈 시간 찾기: 벗 일정 교차체크, 공통 가능 시간 3개 추천
- 🗺️ 동선 계획: 출발장소→약속(도착시간 기준)→복귀, 역산 출발시간 계산
- 약속 제안: 쪽지 발송→수락/거절→자동 일정 등록
- 계산 결과 일정 추가 버튼

## 6. 공유마당 (/share, /share-report)
- 비회원도 공유 가능
- AI 개인정보 검열 (얼굴·번호판→승인보류)
- 동영상 무조건 보류
- 작성자 본인만 미승인글 확인 가능
- 모자이크 처리 버튼
- 이미지 미리보기, 카메라/파일 업로드 통합
- 자세히보기 새탭 열기

## 7. 소식 관리 (/admin/news)
- 원문보기 새탭, AI 번역 버튼
- 편집 페이지에서 원문번역 자동 채우기

## 8. API 연동 현황
| API | 상태 |
|-----|------|
| Kakao Local | 작동 중 (지오코딩/역지오코딩) |
| 문화재청 | 작동 중 (국가유산) |
| Groq AI | 작동 중 (통벗 채팅·번역·교정) |
| 경기도 교통정보 | 키 등록 완료, Code7 대기 |
| 경찰청 UTIC | IP 등록 대기 |
| KEC 도로공사 | VMS 작동, 내용 API 미발견 |
| PortOne 결제 | 테스트모드 구현 완료 |
